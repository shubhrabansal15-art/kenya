Dataset
Source & Coverage
Our World in Data · World Bank · Global Carbon Project · 2002–2022

This dashboard draws on publicly available macro-environmental data compiled primarily from Our World in Data (OWID), which aggregates emissions data from the Global Carbon Project (GCP), and economic data from the World Bank Development Indicators. The dataset covers 20 years (2002–2022), selected for cross-source completeness, and spans Kenya, all African countries with sufficient data coverage, and global benchmarks.

Data Processing
Processing Pipeline
cleaning · transformation · validation
01
Type Coercion
All numeric fields converted using pd.to_numeric(errors='coerce'). Non-parseable values become NaN and are handled downstream.
02
Missing Value Handling
Rows with missing values in key fields are dropped per-calculation rather than globally, preserving as much data as possible for each chart. Time-series charts fill NaN with zero to avoid line breaks.
03
Zero Exclusion for Log Regression
Observations where gdp_per_capita or co2_per_capita ≤ 0 are excluded before log transformation, since log(0) is undefined.
04
Infinity Removal
Elasticity and efficiency metrics can produce ±∞ when divisors approach zero. These are replaced with None before serialisation to avoid frontend rendering errors.
05
Year Filtering
Cross-sectional comparisons filter by a single year. Time-series endpoints default to 2002-2022 but accept user-defined ranges via the slider controls on each page.
Derived Metrics
Four Advanced Metrics
formulas · purpose · interpretation · full definitions in Methodology
01  Decoupling Index
Decoupling Index = ΔGDP (%) − ΔCO₂ (%)

Measures whether GDP growth exceeds emissions growth year-on-year. Positive = relative decoupling. Negative = coupled growth. Used as the primary short-run indicator.

02  Emission Elasticity
ε = (%ΔCO₂) / (%ΔGDP)

Captures emissions sensitivity to economic growth. ε < 1 is relative decoupling; ε < 0 is strong decoupling. High volatility is expected in small open economies.

03  Regression Residual
log(CO₂/cap) = α + β · log(GDP/cap) + ε

Cross-country log-log OLS regression. The residual ε is Kenya's deviation from the expected emissions curve. Negative = emits less than expected for its income. Converted to % via exp(ε) − 1.

04  CAGR Decoupling Score
CAGR = (End/Start)^(1/n) − 1  ·  Score = GDP CAGR − CO₂ CAGR

Long-run structural metric. Smooths year-on-year volatility. A positive score over 20 years is the strongest evidence of sustained relative decoupling.

Assumptions & Limitations
Assumptions
modelling choices · simplifications
  • GDP is estimated as gdp_per_capita × population where a standalone GDP column is absent.
  • CO₂ figures cover fossil fuels, cement and flaring only — land-use change emissions are excluded.
  • GDP per capita is in constant 2015 USD; no PPP adjustment is applied unless noted.
  • Countries are treated independently — trade-adjusted or consumption-based emissions are not modelled.
  • The log-log regression assumes a globally consistent income–emissions relationship.
  • Kenya is always excluded from its own regression training set to avoid self-prediction bias.
Limitations
scope · data gaps · interpretation caveats
  • Data quality and reporting standards vary across countries and years, particularly for smaller African economies.
  • Informal economic activity — significant in Kenya — is not captured in official GDP figures.
  • The energy mix (renewables vs. fossil) is not explicitly modelled; it is inferred from emissions trends.
  • Short-term volatility driven by drought, policy shifts, or commodity cycles can distort year-on-year metrics.
  • CAGR is sensitive to the chosen start and end years; a single anomalous year can distort long-run readings.
  • Regression R² reflects fit across all countries; the residual for a single country carries wider confidence intervals.
Conceptual Framework
Theoretical Foundations
Kaya Identity · Environmental Kuznets Curve · IPCC frameworks
Kaya Identity
CO₂ = Pop × (GDP/Pop) × (E/GDP) × (CO₂/E)

Decomposes emissions into population, income, energy intensity, and carbon intensity of energy. Useful for identifying which driver dominates Kenya's emissions trajectory.

Environmental Kuznets Curve
Emissions ↑ then ↓ as income rises

Hypothesises that pollution first rises then falls as countries develop. Kenya's position on this curve still on the upward slope is tested via the regression residual analysis.

IPCC Carbon Intensity
Intensity = CO₂ / GDP

IPCC mitigation scenarios use carbon intensity as a key tracking metric. Falling intensity alongside rising GDP is the definition of relative decoupling used in this dashboard.
